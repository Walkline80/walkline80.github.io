图片上写的很清楚这个服务器是for PC/Mac的，请不要误会为其它版本的服务器啊

其实方法就和上图说的一样简单

  1. 下载服务器文件
  2. 运行服务器文件

只不过还需要配置一些参数，为了再简化这个过程我写了一个小工具，集成了 启动/停止服务器、配置文件编辑器 和 快捷tp工具 等功能

1、下载服务器文件

服务器文件下载地址在这里，不过官网只能下载最新版本的（和游戏版本同步），所以如果你想要玩之前版本的还需要自行搜索下载

2、下载Java运行库

判断你的系统是否安装了java运行库的方法是：在dos窗口输入 java 并回车，如果没有出现与下面类似的内容就说明需要下载安装运行库

也就是jre，根据你使用的Windows选择x32或x64版本的，建议下载jre1.8

还有一个方法，就是通过官方启动器进行游戏的下载和运行环境的安装，一键搞定！
