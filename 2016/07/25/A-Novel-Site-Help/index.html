小说管理模块使用说明

目录
  1. 简介
  2. 使用方法
    1. 添加书籍
    2. 更新、删除书籍
    3. 完成
  3. 导入书籍管理

简介

目前，管理模块支持添加 顶点中文网、天天中文网、免费小说网 和 八零电子书 4个小说网站的资源(八零电子书不稳定，已屏蔽），共有5个功能，分别是：

  • 查询书籍
  • 添加书籍
  • 更新书籍
  • 删除书籍
  • 导入书籍管理

使用方法

下面就以 顶点中文网 为例介绍一下使用方法

添加书籍

1、点击下图中的粗体字 顶点中文网 或打开 http://m.dingdianzw.com

2、选择一本喜爱的小说，点击打开它的首页

重点：记录首页链接中的数字编号（本例中为2935） http://m.dingdianzw.com/wapbook/2935.html

3、把小说编号填写到管理模块对应空白处并点击 查询 按钮，如果可以查询到正确的信息，管理模块会自动填写其余空白处，按 添加 / 更新 按钮就完成了添加书籍的操作

切记：如果要添加 天天中文网、免费小说网 或 八零电子书 的小说资源，务必将 小说来源 修改为相应的网站

更新、删除书籍

从管理模块下方的小说列表里点击要更新（删除）的小说，信息会自动填写到上边的空白处，这时候：

  • 如果需要更新则直接编辑信息，然后点击 添加 / 更新 按钮完成更新操作
  • 如果需要删除小说则直接点击 删除 按钮即可完成删除操作

完成

完成上述操作后就可以点击 返回 按钮开始阅读喜欢的小说了
